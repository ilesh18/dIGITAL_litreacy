#  Digital Literacy Awareness Quiz

## Project Overview
This project is a **Digital Literacy Awareness Quiz** created using **Google Forms**. The purpose of this quiz is to assess and improve awareness about safe, responsible, and effective use of digital technologies among students.
---
## Platform Used
- **Google Forms**
- https://forms.gle/kKM8sc549xvQJe6N6
  - Used to design and create the quiz
  - Collect responses in an organized manner
  - Enable quiz features like automatic evaluation and required responses
---
## What I Built
I created a quiz consisting of **5 questions**, including:
- Multiple-choice questions
- Checkbox questions
- Short answer questions  

The quiz covers topics such as:
- Understanding digital literacy  
- Safe online practices  
- Password security  
- Importance of digital awareness
- response page- https://docs.google.com/forms/d/1Nd4wYjKflFgsOGudnXsKdjSygOEZOcJHSDLETSlwE2A/edit?pli=1#responses
